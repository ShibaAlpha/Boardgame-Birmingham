## Game Rules

### Goal
Build industries and connections to score the most victory points.

### Actions
1. **Build** - Construct industries in cities
2. **Link** - Build canal/rail connections
3. **Develop** - Upgrade or flip industries
4. **Sell** - Sell goods for income
5. **Loan** - Take loans for starting capital
6. **Scout** - Draw new location cards

### Structure
- Two eras: Canal Era â†’ Rail Era
- 2 actions per turn
- Score at end of each era
